Project Highlights

Semantic Graph API Library

An open source API providing developers the ability to implement a relational database to semantic graph mapping feature to their bespoke software. Utilises W3C standardised semantic web specifications for users to seamlessly configure what data to map from the database input. Such tranformation allows the data to be used in further advance inference and graph analytics to uncover new information and value.

  • Designed and implemented API library to map relational databases to a semantic graph model.
  • Implemented Resource Descriptive Framework (RDF) and RDB to RDF Mapping Language (R2RML) specifications provided publicly by World Wide Web Consortium (W3C).
  • Deployed continuous integration and delivery strategies to increase test automation, rapid development and deployment of key features to public repositories.
